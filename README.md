# GroundLang
Ground is a simple, yet powerful language used to create basic applications.

### Variables
Variables are assigned via `var`. A variable can hold one of 3 types of values: Boolean (true/false), integers, floats, and strings.
> `var age = 26`
> `var price = 12.99`
> `var name = "Jeffery"`
> `var enrolled = True`
